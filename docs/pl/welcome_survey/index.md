# Ankieta powitalna

*Data utworzenia: 2024-09-22*

## Opis projektu

Celem projektu było stworzenie aplikacji, która pozwoliłaby na proste filtrowanie i przeglądanie danych z przykładowej ankiety powitalnej (dane zostały odpowiednio zanonimizowane). Celem aplikacji było utrwalenie komponentów z biblioteki Streamlit oraz zapoznanie się z właściwym zarządzaniem stanem aplikacji (`st.session_state`), tak aby wszystkie przyciski i interakcje były względem siebie responsywne.

## Główne funkcjonalności

- Możliwość przeglądania anonimowej ankiety w celu zapoznania się z interfejsem Streamlit
- Różne typy filtrów pozwalają na bardziej szczegółowe poznanie analizowanych danych
- Zawarte są również wizualizacje
- Aplikacja zawiera także ciekawe wizualizacje, a dla najbardziej wytrwałych - czekają ciekawostki!

## Umiejętności

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- Boto3
