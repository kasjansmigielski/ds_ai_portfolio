# Mój Chatbot

*Data utworzenia: 2024-09-15*

## Opis projektu

Celem projektu było stworzenie własnej wersji Chat GPT, opartej na interfejsie aplikacji Streamlit. Chatbot może przyjąć dowolną osobowość, aby maksymalizować funkcjonalność według naszych preferencji.

## Główne funkcjonalności

- Chatbot pamięta rozmowy i zapisuje je w strukturze plików JSON, a Ty możesz łatwo przełączać się między rozmowami (bez utraty historii czatu)
- Możesz wybierać między różnymi modelami od OpenAI
- Koszty korzystania z AI są liczone
- Możesz nadać chatbotowi indywidualną świadomość, która będzie kierować typami odpowiedzi na zadawane pytania

## Umiejętności

- Python
- Langfuse
- OpenAI
- Streamlit
